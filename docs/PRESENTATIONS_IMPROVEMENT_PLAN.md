## Общая методология для каждой презентации

### Этап 1: Анализ и проверка
1. **Проверка текущей информации**
   - Анализ всех текстов, цифр, метрик
   - Проверка актуальности данных
   - Выявление пробелов в информации

2. **Сверка с документами проекта**
   - White-Paper-DAO-VOD-6.txt / VOD-7.txt
   - VODeco_Complete_Documentation.md
   - TOKENOMICS_V2.md
   - Описание_экосистемы_VODeco.md
   - objects for invest.txt
   - water_economic_model.txt
   - другие релевантные документы

3. **Сверка с данными из интернета**
   - Актуальные статистики водного кризиса
   - Данные ООН, World Bank, UNESCO
   - Рыночные данные (TAM/SAM/SOM)
   - Технологические тренды

### Этап 2: UX/UI проработка
1. **Визуальная иерархия**
   - Правильное использование типографики
   - Контрастность и читаемость
   - Цветовая схема (cyan-glow, ocean-deep)
   - Spacing и layout

2. **Интерактивность**
   - Плавные анимации (framer-motion)
   - Hover эффекты
   - Переходы между слайдами/секциями
   - Responsive дизайн

3. **Визуальные элементы**
   - Иконки (lucide-react)
   - Графики и диаграммы
   - Иллюстрации и схемы
   - 3D элементы (где уместно)

### Этап 3: Дополнение информацией
1. **Расширение контента**
   - Детальные описания
   - Дополнительные метрики
   - Истории успеха
   - Кейсы использования

2. **Попап-окна с подробной информацией**
   - Модальные окна для каждого блока
   - Расширенные описания
   - Визуальные материалы (графики, схемы)
   - Ссылки на источники
   - Интерактивные элементы

### Этап 4: Проверка и оптимизация
1. **Техническая проверка**
   - Нет ошибок в консоли
   - Все ссылки работают
   - Изображения загружаются
   - Анимации плавные

2. **Контентная проверка**
   - Нет опечаток
   - Все цифры корректны
   - Информация актуальна
   - Стиль единообразен

## УНИВЕРСАЛЬНЫЕ КОМПОНЕНТЫ

### InfoPopup Component
```tsx
interface InfoPopupProps {
  title: string;
  content: React.ReactNode;
  trigger: React.ReactNode;
  size?: 'sm' | 'md' | 'lg' | 'xl';
  position?: 'top' | 'bottom' | 'left' | 'right' | 'center';
}
```

### StatisticCard Component
```tsx
interface StatisticCardProps {
  value: string | number;
  label: string;
  description?: string;
  source?: string;
  trend?: 'up' | 'down' | 'stable';
  popupContent?: React.ReactNode;
}
```

### DataSource Component
```tsx
interface DataSourceProps {
  organization: string;
  year: number;
  url?: string;
  data: string;
}
```

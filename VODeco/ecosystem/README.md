# Файлы экосистемы VODeco

## Структура папки

- `contracts/` - смарт-контракты
  - `token/` - токен VODeco
  - `governance/` - контракты управления
  - `staking/` - контракты стейкинга
  - `voting/` - контракты голосования

- `docs/` - документация
  - `whitepaper.md` - техническое описание проекта
  - `tokenomics.md` - экономика токена
  - `governance.md` - система управления
  - `api/` - документация API

- `scripts/` - скрипты для работы с контрактами
  - `deploy/` - скрипты деплоя
  - `test/` - скрипты тестирования
  - `utils/` - вспомогательные скрипты

- `tests/` - тесты контрактов
  - `token/` - тесты токена
  - `governance/` - тесты управления
  - `staking/` - тесты стейкинга
  - `voting/` - тесты голосования
